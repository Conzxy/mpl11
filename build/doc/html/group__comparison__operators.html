Detailed Description

Compares several types with a given operator.

Comparison operators are standard tag-dispatched metafunctions, whose semantics and default implementation share a common pattern described here.

Semantics and default implementation

Let F be a comparison operator representing a C++ operator Op.

When invoked with arguments T1, T2, ...Tn, equivalent to and_<F<T1, T2>, F<T2, Tn...>>.

When invoked with arguments T1 and T2, the semantics and default implementation depend on the operator.

Warning
Differences from the original MPL:
  • Standard tag dispatching is used with these metafunctions.
  • More than two types can be compared at once.
